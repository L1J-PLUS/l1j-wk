Ubuntu 12.04
=====

Setup Guide
-----

Login as root

    sudo su
    
Get the Git

    apt-get update && apt-get -y upgrade && apt-get install -y git subversion
    git clone git@github.com:L1J-PLUS/l1j-wk.git
    
Initialize Ubuntu 12.04

    cd l1k-wk
    chmod +x * && chmod +x */*
    ./Ubuntu_12.04/init.sh
    
Start Server

    ./start.sh
